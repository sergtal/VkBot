# VK + Telegram AutoBot

Бот автоматически:

- Проверяет новые посты в группе ВК
- Ставит лайк
- Делает репост
- Оставляет случайный комментарий
- Присылает отчёт в Telegram

Также Telegram-бот позволяет управлять процессом и редактировать комментарии.

---

## Установка

1. Установить Python 3.11+
2. Скачать проект
3. В терминале перейти в папку проекта

Установить зависимости:

pip install -r requirements.txt

---

## Настройка

Открыть файл config.py

Вставить:

- VK_TOKEN
- TG_TOKEN
- USER_ID
- GROUP_ID

---

## Запуск

python bot.py

---

## Управление

В Telegram написать:

/start

Появится панель управления.

---

## Файлы

comments.txt — список комментариев  
processed_posts.db — база обработанных постов  

Файл базы можно удалить — он создастся автоматически.
